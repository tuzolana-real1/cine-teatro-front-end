Guia de Desenvolvimento Front‑end (React + Vite + Tailwind + Axios)Estrutura de Pastasrc/                        # Pasta principal do código fonte ├── assets/                # Onde ficam imagens, ícones e logos ├── components/            # Componentes reutilizáveis (blocos visuais) │    ├── Header.jsx        # Cabeçalho do site (menu, logo, navegação) │    ├── Footer.jsx        # Rodapé do site (informações finais) │    ├── CardObra.jsx      # Card para mostrar filmes/peças (cartaz + info) │    ├── ModalUpload.jsx   # Modal para upload de vídeo/cartaz │    ├── Button.jsx        # Botão reutilizável com estilo padrão │    └── Input.jsx         # Campo de input reutilizável para formulários │ ├── pages/                 # Páginas principais do site │    ├── Home.jsx          # Página inicial (lista de obras) │    ├── Login.jsx         # Página de login (formulário de email/senha) │    ├── Cadastro.jsx      # Página de cadastro (novo utilizador) │    ├── Dashboard.jsx     # Área do estúdio (gestão de obras/publicações) │    └── PublicarObra.jsx  # Página para publicar nova obra (cartaz + data) │ ├── context/               # Gestão de estado global │    └── AuthContext.jsx   # Contexto de autenticação (login/logout + token) │ ├── services/              # Configuração da API │    └── api.js            # Axios configurado com baseURL e funções padrão │ ├── App.jsx                # Define rotas principais com React Router ├── main.jsx               # Ponto de entrada da aplicação React └── index.css              # Estilos globais + importação do TailwindConvencoes de NomesComponentes: PascalCase → CardObra, ModalUpload, InputVariáveis de estado: camelCase → userData, isLoggedIn, conteudosFunções de API: verbo + recurso → getConteudos(), createObra(), updateVideo()Contexto de autenticação: login(token), logout(), userRotas Principais(App.jsx)<Routes>                                   // Define todas as rotas da aplicação  <Route path="/" element={<Home />} />    // Página inicial  <Route path="/login" element={<Login />} /> // Página de login  <Route path="/cadastro" element={<Cadastro />} /> // Página de cadastro  <Route path="/dashboard" element={<Dashboard />} /> // Área do estúdio  <Route path="/publicar" element={<PublicarObra />} /> // Publicar nova obra</Routes>API(services/api.js)import axios from "axios";                           // Importa biblioteca axiosconst api = axios.create({                           // Cria instância axios  baseURL: "http://localhost:8080/api",              // URL base do backend});// Funções padrão para chamadas à APIexport const getConteudos = () => api.get("/conteudos");               // Buscar lista de obrasexport const createObra = (data) => api.post("/conteudos", data);      // Criar nova obraexport const updateVideo = (id, videoUrl) =>                           // Atualizar vídeo de obra  api.put(`/conteudos/${id}/video`, { videoUrl });export const registerUser = (data) => api.post("/auth/register", data); // Cadastro de utilizadorexport const loginUser = (data) => api.post("/auth/login", data);       // Login de utilizadorexport default api;                                                    // Exporta instânciaFluxo de trabalho com git1. Push para GitHub → git push origin feature/login-page4. Abrir Pull Request → revisão → merge na main
